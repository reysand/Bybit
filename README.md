## Tech stack
- Minimum SDK level 33
- [Kotlin](https://kotlinlang.org/) based
- Jetpack
    - Compose - A modern toolkit for building native Android UI.
    - ViewModel - UI related data holder, lifecycle aware.
- Architecture
    - MVVM Architecture (View - ViewModel - Model)
- Ktor - HTTP client
